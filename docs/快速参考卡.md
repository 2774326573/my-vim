# Win7 无网环境 - 快速参考卡

## 🚀 启动 Vim

```powershell
# Windows PowerShell
.\scripts\windows\run_portable_windows.ps1

# Windows CMD（无 PowerShell 权限）
scripts\windows\run_portable_windows_nops1.cmd
```

## 📋 LSP 功能快捷键

### 代码导航
| 快捷键 | 功能                | 适用语言 |
| ------ | ------------------- | -------- |
| `gd`   | 跳转到定义          | 所有 LSP |
| `gD`   | 跳转到定义（同 gd） | 所有 LSP |
| `gr`   | 查找引用            | 所有 LSP |
| `gi`   | 跳转到实现          | 所有 LSP |
| `gy`   | 跳转到类型定义      | 所有 LSP |
| `K`    | 显示悬停文档        | 所有 LSP |

### 代码编辑
| 快捷键       | 功能       |
| ------------ | ---------- |
| `<leader>rn` | 重命名符号 |
| `<leader>ca` | 代码操作   |
| `<leader>cf` | 格式化代码 |

### 诊断信息
| 快捷键       | 功能         |
| ------------ | ------------ |
| `[d`         | 上一个诊断   |
| `]d`         | 下一个诊断   |
| `<leader>cd` | 显示所有诊断 |
| `<leader>cs` | 显示文档符号 |

### 代码补全
| 快捷键       | 功能            |
| ------------ | --------------- |
| `Ctrl+Space` | 手动触发补全    |
| `Tab`        | 选择下一项      |
| `Shift+Tab`  | 选择上一项      |
| `Enter`      | 确认选择        |
| 自动触发     | 输入 2 个字符后 |

### LSP 状态检查
```vim
:LspStatus          " 查看所有 LSP 服务器状态
:LspInfo            " 详细 LSP 信息
:LspLog             " 查看 LSP 日志
```

## 🎨 前端开发专用

### Emmet 快速编写
| 快捷键       | 功能            | 示例                                      |
| ------------ | --------------- | ----------------------------------------- |
| `<leader>e,` | 展开 Emmet 缩写 | `div.class` → `<div class="class"></div>` |

**Emmet 示例：**
```html
<!-- 输入后按 <leader>e, -->
ul>li*3>a

<!-- 展开为： -->
<ul>
  <li><a href=""></a></li>
  <li><a href=""></a></li>
  <li><a href=""></a></li>
</ul>
```

| 标签高亮    | 自动高亮匹配的开/闭标签 |
| CSS 颜色    | 自动显示颜色预览        |

## 💡 提示

- `<leader>` 键默认是 `Space`（空格）
- 所有功能均可离线使用
- C++、Python、前端三件套完全支持
- Win7 环境使用 Node.js 12 兼容版语言服务器
- 离线翻译需要 StarDict 词典
