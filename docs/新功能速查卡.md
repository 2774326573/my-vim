# 新功能速查卡

## 🎨 彩虹括号

**自动启用** - 不同层级的括号显示不同颜色

| 命令             | 说明         |
| ---------------- | ------------ |
| `:RainbowToggle` | 切换彩虹括号 |

**效果**:
```cpp
void func() {        // 第1层 - 蓝色
    if (x > 0) {     // 第2层 - 橙色
        foo({        // 第3层 - 绿色
            bar()    // 第4层 - 红色
        })
    }
}
```

## 📁 代码折叠

### 快捷键

| 快捷键       | 功能         |
| ------------ | ------------ |
| `<Space>zz`  | 切换当前折叠 |
| `<Space>zm`  | 关闭所有折叠 |
| `<Space>zr`  | 打开所有折叠 |
| `<Space>zo`  | 打开当前折叠 |
| `<Space>zc`  | 关闭当前折叠 |
| `<Leader>uf` | 切换折叠功能 |

### 原生快捷键

| 快捷键 | 功能           |
| ------ | -------------- |
| `za`   | 切换当前折叠   |
| `zA`   | 递归切换       |
| `zo`   | 打开折叠       |
| `zO`   | 递归打开       |
| `zc`   | 关闭折叠       |
| `zC`   | 递归关闭       |
| `zM`   | 关闭所有       |
| `zR`   | 打开所有       |
| `zj`   | 移到下一个折叠 |
| `zk`   | 移到上一个折叠 |

### 配置

```vim
set foldmethod=indent  " 基于缩进折叠
set foldlevelstart=99  " 打开文件时不折叠
set nofoldenable       " 默认禁用折叠
```

## 🗄️ 数据库插件

### 快捷键

| 快捷键       | 功能              |
| ------------ | ----------------- |
| `<Space>db`  | 打开/关闭数据库UI |
| `<Space>dba` | 添加数据库连接    |
| `<Space>dbf` | 查找数据库缓冲区  |
| `<Space>dbr` | 重命名缓冲区      |
| `<Space>dbl` | 查看最后查询信息  |

### 数据库UI操作

在数据库UI窗口中：
- `o` 或 `<CR>` - 展开/折叠
- `S` - 选择连接
- `R` - 重新加载
- `<Leader>S` - 执行选中的SQL
- `<Leader>SS` - 执行整个文件

### 连接字符串示例

```vim
" MySQL
mysql://root:password@localhost/mydb

" PostgreSQL
postgresql://user:pass@localhost:5432/mydb

" SQLite
sqlite:///path/to/database.db

" MongoDB
mongodb://localhost:27017/mydb
```

### 执行SQL

```vim
" 执行选中的SQL
:'<,'>DB mysql://localhost/mydb

" 执行整个文件
:%DB

" 使用变量
:let g:db = 'mysql://localhost/mydb'
:'<,'>DB g:db
```

## 📝 UltiSnips 代码片段 (已优化)

### 触发键

| 快捷键      | 功能                            |
| ----------- | ------------------------------- |
| `Tab`       | 展开代码片段 (补全菜单不可见时) |
| `Tab`       | 跳到下一个占位符                |
| `Shift+Tab` | 跳到上一个占位符                |
| `Ctrl+L`    | 列出可用片段                    |

### 使用流程

1. **输入片段触发词**: 例如 `main`
2. **按 Tab**: 展开为完整代码
3. **编辑占位符**: 修改高亮的占位符
4. **按 Tab**: 跳到下一个占位符
5. **完成**: 所有占位符填完后退出片段模式

### 示例

#### C++ 代码片段

```cpp
// 输入: main<Tab>
int main() {
    // TODO
    return 0;
}

// 输入: for<Tab>
for (size_t i = 0; i < count; i++) {
    // TODO
}

// 输入: class<Tab>
class ClassName {
private:
    // members
public:
    ClassName();
    ~ClassName();
};
```

#### Python 代码片段

```python
# 输入: def<Tab>
def function_name(args):
    """docstring"""
    pass

# 输入: class<Tab>
class ClassName:
    """docstring"""
    def __init__(self):
        pass
```

#### HTML 代码片段

```html
<!-- 输入: html5<Tab> -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

### 自定义片段

编辑片段文件: `<Leader>ue`

位置: `mysnippets/filetype.snippets`

## 🔖 标记可视化 (vim-signature)

### 快捷键

| 快捷键     | 功能                    |
| ---------- | ----------------------- |
| `m,`       | 放置/删除下一个可用标记 |

| `:marks`    | 列出所有标记      |

## 🎯 补全优化

### 智能Tab键

- **补全菜单可见**: Tab 选择下一项
- **补全菜单不可见**: 
  - 如果可以展开代码片段: 展开
  - 否则: 插入Tab字符

### 回车键

- **补全菜单可见**: 接受选中项并关闭菜单
- **补全菜单不可见**: 正常换行

### 手动触发补全

- `Ctrl+Space`: 手动触发补全菜单

## 🌐 HTML 实时预览

### 快捷键

| 快捷键      | 功能           |
| ----------- | -------------- |
| `<Space>hp` | 启动 HTML 预览 |
| `<Space>hs` | 停止 HTML 预览 |
| `<Space>ht` | 切换 HTML 预览 |

### 使用方式

1. 打开并保存一个 `.html` 或 `.htm` 文件
2. 按 `<Space>hp` 启动浏览器预览
3. 后续每次保存文件，浏览器会自动刷新

### 说明

- 预览服务使用仓库自带的 `tools/python38`
- 预览地址会在命令行消息区显示
- 当前实现监听的是文件保存后的变更，不会同步未保存内容
