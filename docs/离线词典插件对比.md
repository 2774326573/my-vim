## 🎯 1. vim-translator (推荐首选)

### 基本信息
- **GitHub**: https://github.com/voldikss/vim-translator
- **作者**: voldikss
- **Stars**: ~1.4k
- **最后更新**: 活跃维护中

### 离线配置

#### 步骤 1: 安装插件
```vim
Plug 'voldikss/vim-translator'
```

#### 步骤 2: 下载 sdcv (Windows)
- **下载地址**: https://sourceforge.net/projects/sdcv/
- 或者编译版本: https://github.com/Dushistov/sdcv/releases
- 将 `sdcv.exe` 放到 `tools/sdcv/` 目录

#### 步骤 3: 下载 StarDict 词典
推荐词典源:
- **ECDICT StarDict版**: https://github.com/skywind3000/ECDICT
- **stardict-langdao**: https://github.com/huzheng001/stardict-3
- **牛津高阶**: 搜索 "stardict oxford" 下载

词典文件结构:
```
C:\Users\jinxi\my-vim\data\dicts\
├── stardict-ecdict/
│   ├── ecdict.dict.dz
│   ├── ecdict.idx
│   └── ecdict.ifo
├── langdao-ec-gb/
│   ├── langdao-ec-gb.dict.dz
│   ├── langdao-ec-gb.idx
│   └── langdao-ec-gb.ifo
```

#### 步骤 4: 配置 Vim
```vim
" config/plugin-config.vim
" =============== vim-translator 离线翻译 ===============
let g:translator_default_engines = ['sdcv']
let g:translator_window_type = 'popup'  " 或 'preview'
let g:translator_target_lang = 'zh'

" 设置 sdcv 路径和词典目录 (Windows)
if has('win32') || has('win64')
  let g:translator_custom_binary = myvim#path#join(g:my_vim_config_dir, 'tools', 'sdcv', 'sdcv.exe')
  let $STARDICT_DATA_DIR = myvim#path#join(g:my_vim_config_dir, 'data', 'dicts')
endif

" 快捷键
nmap <Leader>tw <Plug>TranslateW
vmap <Leader>tw <Plug>TranslateWV
nmap <Leader>tr <Plug>TranslateR
vmap <Leader>tr <Plug>TranslateRV
```

### 优点
✅ 弹窗显示翻译结果，不干扰编辑  
✅ 支持光标下单词或选中文本翻译  
✅ 支持在线引擎 (bing/google) 和离线 (sdcv) 切换  
✅ StarDict 格式兼容大量词典  
✅ 可以使用 ECDICT 超大词库 (770,000 词条)  

### 缺点
❌ 需要单独下载 sdcv 可执行文件  
❌ 需要手动下载和配置词典文件  
❌ Windows 下路径配置稍复杂

## 🎯 2. vim-dict (已安装，最简单)

### 基本信息
- **GitHub**: https://github.com/skywind3000/vim-dict
- **作者**: skywind3000 (国内作者)
- **Stars**: ~800
- **最后更新**: 活跃维护中

### 使用方法

#### 已有配置
```vim
" config/plugin-config.vim (已有)
let s:vim_dict_repo = myvim#path#join(g:myvim_data_dir, 'plugged', 'vim-dict', 'dict')
let g:vim_dict_dict = get(g:, 'vim_dict_dict', []) + [s:vim_dict_repo]
```

#### 内置词库
插件自带基础英汉词库:
```
data/plugged/vim-dict/dict/
├── c.txt        # C 语言关键字
├── cpp.txt      # C++ 关键字
├── cs.txt       # C# 关键字
├── java.txt     # Java 关键字
├── php.txt      # PHP 关键字
├── python.txt   # Python 关键字
├── ...
```

#### 扩展词库 (使用 ECDICT)
1. 下载 ECDICT: https://github.com/skywind3000/ECDICT/releases
2. 转换为 vim-dict 格式 (简单文本):
```
# 每行格式: 单词 释义
hello 你好
world 世界
...
```
3. 添加到配置:
```vim
let g:vim_dict_dict = [
  \ 'C:/Users/jinxi/my-vim/data/plugged/vim-dict/dict',
  \ 'C:/Users/jinxi/my-vim/data/dicts/ecdict.txt'
  \ ]
```

### 优点
✅ 零依赖，纯 VimScript 实现  
✅ 自动补全集成 (插入模式 `Ctrl-X Ctrl-K`)  
✅ 内置多种编程语言关键字  
✅ 配置极其简单  
✅ Windows 7 完美兼容  

### 缺点
❌ 只支持自动补全，不能弹窗显示翻译  
❌ 内置词库较小 (~16MB)  
❌ 文本格式词典，不如 StarDict 功能丰富

## 🎯 3. ecdict-ultimate (ECDICT 专用)

### 基本信息
- **GitHub**: https://github.com/chxuan/ecdict-ultimate
- **作者**: chxuan
- **Stars**: ~200
- **词库**: 基于 skywind3000/ECDICT (770,000 词条)

### 安装步骤

#### 1. 安装插件
```vim
Plug 'chxuan/ecdict-ultimate', { 'do': 'python install.py' }
```

#### 2. 下载 ECDICT 数据库
- 下载: https://github.com/skywind3000/ECDICT/releases
- 解压 `ecdict.db` (SQLite 数据库，~380MB)
- 放到插件目录: `data/plugged/ecdict-ultimate/`

#### 3. 配置
```vim
" config/plugin-config.vim
" =============== ecdict-ultimate ===============
let g:ecdict_db_path = myvim#path#join(g:myvim_data_dir, 'plugged', 'ecdict-ultimate', 'ecdict.db')

" 快捷键
nnoremap <Leader>ed :EcdictTranslate <C-r><C-w><CR>
```

### 优点
✅ 使用 ECDICT 超大词库 (770,000 词条)  
✅ 包含词频、音标、英英释义  
✅ SQLite 数据库查询速度快  
✅ 支持模糊搜索  

### 缺点
❌ 需要 Python 环境 (Win7 需要 Python 2.7 或 3.x)  
❌ 数据库文件较大 (~380MB)  
❌ Windows 7 下 Python 配置可能有问题

## 🎯 5. sqlite.lua + ECDICT (Neovim only)

### 基本信息
- **GitHub**: https://github.com/kkharji/sqlite.lua
- **仅支持**: Neovim 0.7+
- **不支持**: 传统 Vim

### 配置示例 (Neovim)
```lua
-- init.lua
require('sqlite').setup()
local db = require('sqlite.db')

local ecdict = db:open('path/to/ecdict.db')
function translate_word(word)
  local result = ecdict:select('stardict', {
    where = { word = word }
  })
  print(result[1].translation)
end
```

### 优点
✅ Lua 接口性能优秀  
✅ 直接查询 ECDICT SQLite 数据库  
✅ Neovim 集成度高  

### 缺点
❌ **不支持传统 Vim**  
❌ 需要 Neovim 0.7+  
❌ 配置较复杂

## 📝 ECDICT CSV 转 vim-dict TXT 脚本

```python
# convert_ecdict.py
import csv

with open('ecdict.csv', 'r', encoding='utf-8') as fin:
    with open('ecdict.txt', 'w', encoding='utf-8') as fout:
        reader = csv.DictReader(fin)
        for row in reader:
            word = row['word']
            translation = row.get('translation', '')
            if translation:
                fout.write(f"{word}\t{translation}\n")
```

运行: `python convert_ecdict.py`

## ❓ 常见问题

### Q: sdcv 找不到词典
**A**: 检查环境变量 `$STARDICT_DATA_DIR` 是否正确设置，词典文件是否完整 (`.ifo`, `.idx`, `.dict.dz`)

### Q: vim-dict 补全不生效
**A**: 确认 `g:vim_dict_dict` 路径正确，插入模式按 `Ctrl-X Ctrl-K` 手动触发

### Q: Python 插件在 Win7 无法使用
**A**: 确保安装 Python 2.7 或 3.x，并配置 `g:python3_host_prog` 指向 `python.exe`

### Q: 词典文件太大怎么办
**A**: 
- vim-dict: 只保留需要的语言词典
- ECDICT: 使用 CSV 格式 (~90MB) 而非 SQLite (~380MB)
- StarDict: 选择单个小词典 (50-100MB)
