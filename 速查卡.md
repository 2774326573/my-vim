# Vim 快捷键速查卡 (LazyVim 风格)

## 🎯 核心快捷键 (最常用)

```
Leader = Space (空格键)
```

### ⚡ 超高频操作
| 快捷键 | 功能 | 快捷键 | 功能 |
|--------|------|--------|------|
| `<leader>w` | 保存 | `<leader>q` | 退出 |
| `<leader>e` | 文件树 | `<leader>ff` | 查找文件 |
| `<leader>fg` | 全文搜索 | `<leader><space>` | Buffer 列表 |
| `Shift+h/l` | 切换 Buffer | `Ctrl+h/j/k/l` | 窗口导航 |
| `gd` | 跳转定义 | `gr` | 查找引用 |
| `K` | 悬浮文档 | `<leader>ca` | 代码操作 |
| `gcc` | 注释行 | `<leader>cf` | 格式化 |

### 📂 文件与搜索
```
<leader>ff    查找文件         <leader>fg    全文搜索
<leader>fb    Buffer 列表      <leader>fr    最近文件
<leader>fh    搜索帮助         <leader>fw    窗口列表
<leader>/     Ripgrep 搜索     <leader>:     命令历史
```

### 🪟 窗口管理
```
Ctrl+h/j/k/l  窗口导航         <leader>ws    水平分屏
<leader>wv    垂直分屏         <leader>wd    删除窗口
<leader>-     水平分屏         <leader>|     垂直分屏
Ctrl+方向键   调整窗口大小     <leader>=     平衡窗口
```

### 📑 Buffer & Tab
```
Shift+h       上一个 Buffer    Shift+l       下一个 Buffer
<leader>bd    删除 Buffer      <leader>bo    删除其他
[b / ]b       Buffer 切换      <leader>bb    Buffer 列表

<leader><tab><tab>  新建标签   <leader><tab>d   关闭标签
<leader><tab>]      下一个     <leader><tab>[   上一个
```

### ✏️ 编辑操作
```
gcc           注释行           gc{motion}    注释
Alt+j/k       移动行           > / <         缩进
ga            对齐工具         ysiw"         添加引号
cs"'          引号切换         ds"           删除引号
```

### ⚡ 快速移动
```
J / K         上下 5 行        H / L         行首/尾
s{char}{char} EasyMotion      gg / G        首/尾行
[d / ]d       诊断跳转         [h / ]h       Git Hunk

<leader>cf    格式化           <leader>cd    诊断列表
[d / ]d       诊断跳转         <leader>cs    符号列表
```

## 📝 编辑模式基础

### 模式切换
```
i / a         插入模式         I / A         行首/尾插入
o / O         新行插入         v             可视模式
V             行可视           Ctrl+v        块可视
Esc / Ctrl+[  退出模式         :             命令模式
```

### 基础移动
```
h j k l       左下上右         w / b         单词移动
e             词尾             0 / $         行首/尾
gg / G        首/尾行          %             括号匹配
f{char}       查找字符         t{char}       到字符前
```

### 基础操作
```
x             删除字符         dd            删除行
yy            复制行           p / P         粘贴
u             撤销             Ctrl+r        重做
.             重复操作         /             搜索
n / N         下/上一个        *             搜索单词
```

### 文本对象
```
iw / aw       词内/含空格      i" / a"       引号内/含引号
i( / a(       括号内/含括号    it / at       标签内/含标签
ip / ap       段落内/含段落    is / as       句子内/含句子
```

### 操作组合
```
diw           删除单词         ciw           改变单词
yi"           复制引号内       da(           删除含括号
ci{           改变括号内       =ap           格式化段落
```

## 🎓 学习路径

### 第一天：基础移动
- `h j k l` 方向移动
- `i a o` 插入模式
- `:w :q :wq` 保存退出
- `<leader>e` 文件树
- `<leader>ff` 查找文件

### 第二天：编辑操作
- `dd yy p` 剪切复制粘贴
- `u Ctrl+r` 撤销重做
- `gcc` 注释
- `ciw diw` 改变/删除单词
- `<leader>w <leader>q` 保存退出

### 第三天：代码导航
- `gd` 跳转定义
- `gr` 查找引用
- `K` 悬浮文档
- `Ctrl+o Ctrl+i` 跳转历史
- `<leader>fg` 全文搜索

### 第四天：高级编辑
- `ys cs ds` 包围符
- `Alt+j/k` 移动行
- `ga` 对齐
- `<leader>ca` 代码操作
- `<leader>rn` 重命名

### 第五天：窗口管理
- `Ctrl+h/j/k/l` 窗口导航
- `<leader>ws/wv` 分屏
- `Shift+h/l` Buffer 切换
- `<leader>bd` 删除 Buffer
- `<leader><tab>` Tab 操作

## 💡 Pro 技巧

1. **快速修复**: `gd` 跳转定义后，`Ctrl+o` 返回
2. **多文件编辑**: `<leader>fg` 搜索后，`Tab` 选择多个，`Enter` 打开
3. **批量重命名**: `<leader>rn` 重命名符号
4. **快速注释**: 可视模式选中，`gc` 注释
5. **行移动**: `Alt+j/k` 比剪切粘贴快
6. **多光标**: `Ctrl+n` 选中单词，继续 `Ctrl+n` 选中更多
7. **代码对齐**: 可视选中，`ga=` 按等号对齐
8. **Git 预览**: `<leader>hp` 预览更改，`<leader>hs` 暂存
